# Menu.nyx
Menu for Nyx.ps1 (https://github.com/evilsocket/nyx)

**Eng:**

**nyx_Menu_En.bat** - interactive launch menu for Nyx Cleaner that lets you run dry-run, normal or force clean, advanced wipes (including hiberfil/pagefile), debug mode, and specific anti-forensics modules via nyx.ps1.

**nyx_Menu_Ru.bat** – Russian-localized console menu for Nyx Cleaner providing the same cleaning modes and modular selection, plus quick language switching between English and Russian launchers.

**Rus:**

**nyx_Menu_Ru.bat** — консольное меню запуска Nyx Cleaner на русском языке, позволяющее выбирать режим очистки (тестовый, обычный, принудительный, расширенный), включать подробный вывод и запускать отдельные модули анти‑форензики через nyx.ps1.

**nyx_Menu_En.bat** — англоязычное меню для управления Nyx Cleaner: выбор сухого запуска, нормальной или форсированной очистки, расширенных сценариев (hiberfil/pagefile), режима отладки и запуска конкретных модулей очистки следов.​
